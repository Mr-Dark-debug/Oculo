Introduction

Oculo is a high-performance observability tool built specifically for AI Agents. It provides the "Glass Box" needed to understand non-deterministic agent cognition by capturing thought traces, memory mutations, and prompt evolution in real-time.

Unlike traditional observability (APM) which focuses on request/response and latency, Oculo treats AI-specific data—like context windows and internal reasoning steps—as first-class citizens.

Architecture Overview

Oculo follows a local-first, high-throughput design:

  • 01
    The SDK: A lightweight instrumentation layer in Python or Go that ingests traces over Unix Sockets or TCP.
  • 02
    The Daemon: A Go-based ingestion engine that batches and persists data into a local SQLite store.
  • 03
    The Visualizer: A TUI (Terminal User Interface) or Web-based dashboard to explore thought traces.

Python SDK Usage

Install the SDK via pip:

pip install oculo-sdk

Instrument your agent trace:

from oculo import OculoTracer

tracer = OculoTracer(agent_name="researcher-v1")

with tracer.trace() as trace:
    with trace.span("llm_call", operation_type="LLM") as span:
        span.set_prompt("Who are you?")
        # Agent logic here
        span.set_completion("I am an AI assistant.")
