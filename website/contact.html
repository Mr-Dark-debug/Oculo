FAQ

Common Questions.

Is Oculo free?

Yes. Oculo is fully open source under the MIT license. There are no paid tiers or premium features.

Does Oculo send data to the cloud?

Never. All data stays on your local machine in a SQLite database. There is no telemetry, analytics, or outbound network calls.

What languages are supported?

Currently Python and Go. JavaScript/TypeScript and Rust SDKs are planned for v1.0.

How do I contribute?

Fork the repo, find an open issue, and submit a pull request. We welcome contributions to the daemon, SDKs, TUI, and documentation.
