Architecture

Three-Layer Design.

SDK ‚Üí Daemon ‚Üí Visualizer. Simple, composable, fast.

SDK

Python / Go

Lightweight instrumentation libraries that wrap your agent code in traces and spans. Non-blocking, context-managed.

Engine

Go Daemon

High-throughput ingestion over UDS/TCP. Batches data into a local SQLite store with WAL journaling and FTS5 indexing.

View

TUI / Web

Terminal or browser-based dashboards for exploring traces, diffing memory, and analyzing token patterns in real-time.
