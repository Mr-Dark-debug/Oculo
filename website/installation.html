Up and Running.

Install the Oculo daemon and start tracing your agents in seconds.

Automated Script

The fastest way to install Oculo on macOS, Linux, or WSL. Our script automatically detects your architecture and installs the latest binary.

curl -sSf https://install.oculo.dev | bash

Package Managers

macOS (Homebrew)
brew install oculo-project/tap/oculo
Windows (WinGet)
winget install oculo

Docker

Run the Oculo daemon in a isolated container environment.

docker pull oculodev/oculo:latest
